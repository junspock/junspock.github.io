Markdown is a way to style text on the web. You control the display of the document;formatting words as bold or italic, addinf images , and creating lists are just a few of the things we can do wiht Makrdown. Mostly, Markdown is just regular text with a few non-alphabetic characters thrown in, like # or * .

HEADERS

# This is an <h1> tag
## This is an <h2> tag
###### This is an <h3> tag

This is an <h1> tag

This is an <h2> tag

This is an <h3> tag

EMPHASIS

*This text will be italic*
_This will also be italic_

**This text will be bold**
__This will also be bold__

*You **can** combine them*

Output will be like this :

===========================

This text will be italic

This will also be italic

This text will be bold

This will also be bold

*You can combine them*
